# everylint

Lint your js, css, markdown, html and text just with one linter.

Supports:

* [x] eslint
* [x] stylelint
* [ ] markdownlint
* [ ] htmlhint
* [ ] yaspeller

## Usage

Clone this repo and link

```
npm link
```

In your some project add this

```
npm link everylint
```

Run

```
npx everylint
```
